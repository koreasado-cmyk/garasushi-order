가경점/방서점/전주점 발주 체크리스트 → 카톡으로 바로 전송.

## 이번 버전 변경사항 (v20260706-1200)

1. **iOS 홈 화면 아이콘 수정** — SVG 아이콘은 iOS에서 미지원 → PNG 아이콘 4종 추가
   (`apple-touch-icon.png`, `icon-192/512.png`, `icon-maskable-512.png`)
2. **카톡 200자 잘림 방지** — 카카오 텍스트 템플릿은 200자 초과 시 내용이 잘림.
   긴 발주서는 자동으로 시스템 공유(카톡 선택 가능)로 우회하여 전체 내용 전송
3. **새 버전 자동 업데이트** — 새 버전 배포 시 앱에서 "지금 업데이트?" 확인 후 즉시 적용
   (입력 중인 발주 내용은 자동 저장되어 유지)
4. **작성자 입력 추가** — 날짜 옆 작성자 칸. 점포별로 자동 기억, 발주서 상단에 표기
5. **발주서 하단 합계 건수** 자동 표기
6. **+/− 버튼 소수점 오차 보정** (예: 1.2 + 1 = 2.2000000000000002 방지)
7. **전송 버튼 중복 탭 방지**

## 배포

GitHub Pages에 그대로 올리면 됩니다:
1. GitHub.com 가입/로그인
2. 새 Repository 생성 (이름 예: `garasushi-order`, **Public**)
3. 이 폴더의 **모든 파일** 업로드 (index.html, sw.js, manifest.json, 아이콘 PNG/SVG, logo.png)
4. Repository → Settings → Pages → Source: `main branch` → Save
5. 1~2분 후 `https://[GitHub아이디].github.io/garasushi-order/` 에서 접속 가능

※ 기존 배포에 덮어쓰는 경우에도 새 아이콘 PNG 4개를 반드시 함께 업로드하세요.

## 카카오 연동

1. https://developers.kakao.com 가입/로그인
2. 내 애플리케이션 → 애플리케이션 추가
3. 발급된 **JavaScript 키** 복사
4. 플랫폼 → Web 플랫폼 등록 → 위 GitHub Pages 주소 입력
5. 사이트에서 ⚙ 설정 → JavaScript 키 붙여넣기 → 저장

설정 후 [💬 카톡 전송] 누르면 카톡 채팅 선택 화면이 바로 뜹니다.
(발주서가 200자를 넘으면 잘림 방지를 위해 시스템 공유 시트가 대신 열립니다 — 여기서 카카오톡 선택)

## 사용

1. 점포 선택 (가경/방서/전주) — 마지막 선택 자동 기억
2. 발주일자·작성자 입력 (작성자는 점포별 자동 기억)
3. 품목별 + / − 또는 숫자 입력 (자동저장) — 길게 누르면 ±5
4. 하단 노란 버튼 [💬 발주서 보내기]
5. 미리보기 확인 → [💬 카톡] → 채팅방 선택 → 저장+초기화
